# 4.1 Tipe data

Tipe data dibagi kedalam beberapa bagian. 

- Numerik (desimal, non desimal)
- String
- Boolean


# 4.2 Variabel

Variabel merupakan tempat menyimpan data/value sementara. Setiap variabel disimpan dalam memori. Ingat variabel butuh jenis tipe data (untuk tipe akan dibahas pada [tipe data](#tipe-data)). Dalam go ada beberapa jenis penulisan variabel. 

### 4.2.1 ```var```

Tulis variabel dengan kata kunci ```var```. Lihat contoh penulisan berikut: 

```go
var firstName string = "Agus";

```
atau deklarasi tanpa jenis tipe variabel.

```go
var lastName;

lastName := "Stiawan";

```
